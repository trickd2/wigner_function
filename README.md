# Representación en espacio de fase con la función de Wigner

Código y figuras para representar estados cuánticos (oscilador armónico, pozo infinito y parte radial del hidrógeno) mediante la función de Wigner. Incluye scripts/notebooks, imágenes de las gráficas y referencias completas (formato IEEE).
